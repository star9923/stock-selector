## 限制说明

### ⚠️ 不适合作为选股主数据源

**原因**：
1. 缺少涨跌幅数据 - 无法判断涨跌停
2. 缺少成交量数据 - 无法判断停牌
3. 缺少换手率数据 - 无法进行流动性分析
4. 缺少估值数据 - 无法进行基本面分析

**测试结果**：
```
✅ 获取到 5489 只股票
❌ 基础过滤后剩余 0 只（因为缺少关键字段）
```

### ✅ 适合作为补充数据源

**优势**：
1. 提供社区热度（关注人数）
2. 连接稳定，非交易时间也可用
3. 数据量大（5000+只股票）

**适用场景**：
- 获取股票关注度排行
- 社区情绪分析
- 热门股票筛选
